# Changelog

本文件記錄《紅樓夢知識平台》的主要版本更新。版本號用於標示 demo site、資料流程與數位人文方法展示的階段性變更。

## v0.3.0 - 人物關係圖重構

- 新增 D3 樹狀人物關係圖。
- 建立家族、婚姻、主僕、旁系、姻親與其它人物區塊。
- 調整賈史王薛等家族的樹狀呈現規則。
- 新增人物權威表、親屬表、婚姻表與僕役關係表的正規化流程。
- 新增賈桂「後四十回」標記，區分續書來源與正文主幹關係。
- 簡化人物關係圖控制欄，只保留家族區塊定位。

## v0.2.0 - 人物權威表與 NER Pipeline

- 建立 `person_authority.csv` 與 `person_alias.csv`。
- 將人物權威資料同步至 NER 權威表與別名表。
- 新增人物出現摘要與人物社會網絡資料。
- 調整 `DATA_SCHEMA.md` 為公開 repo 適用的概念層資料結構說明。

## v0.1.0 - 基礎文本平台與共現分析

- 建立章回、段落、句子、token 等 corpus tables。
- 建立全文閱讀、檢索與統計頁面。
- 建立實體共現圖與人物社會網絡圖。
- 建立 `site/` 與 `demo/` 靜態網站輸出流程。
