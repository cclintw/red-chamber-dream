# Citation

This file provides the recommended citation format, research attribution guidance, and methodological citation scope for the Red Chamber Dream Knowledge Platform. The README keeps only a brief citation note; the fuller citation guidance is maintained here.

## Recommended Citation

If you use, cite, adapt, or refer to this project's methodology, data workflow, table design, annotation process, knowledge graph architecture, website presentation, or code in a paper, research project, teaching material, website, software, database, digital humanities platform, or derived system, please cite this project.

Suggested citation:

```text
Lin, Chance. Red Chamber Dream Knowledge Platform: A Digital Humanities Workflow and Knowledge Graph Demonstration for Classical Chinese Fiction. GitHub repository, 2026.
https://github.com/cclintw/red-chamber-dream
```

## Research Attribution

This project is a digital humanities workflow demonstration platform. If you cite or adapt its methodology, workflow, schema, annotation pipeline, knowledge graph design, or static website architecture, please acknowledge the source.

The project demonstrates, among other things:

- A corpus construction workflow from plain text to chapters, paragraphs, sentences, and tokens.
- A named entity recognition workflow combining authority tables, alias tables, and rule tables.
- Analysis of characters, places, buildings, identities, motifs, and co-occurrences in a literary text.
- Methods for producing character relationship graphs, co-occurrence networks, and statistical summaries.
- Conversion workflows among CSV, SQLite, JSON, and static websites.
- Static digital humanities website architecture.
- Annotation pipeline design, schema design, knowledge graph design, and hybrid workflow design.

If you adapt this project, consider adding a note such as:

> Part of the data workflow, annotation pipeline, schema design, or knowledge graph presentation is adapted from Chance Lin's "Red Chamber Dream Knowledge Platform" project, a digital humanities workflow demonstration for classical Chinese fiction.

## Methodological Scope

This project is not only a website about Dream of the Red Chamber. It is also a reusable digital humanities workflow for classical Chinese fiction. When citing it, please describe what you used:

- If you use the code, cite the repository.
- If you use the tables or schema, cite the data workflow and schema design.
- If you use the annotation method, cite the annotation pipeline and hybrid workflow.
- If you use the character network or co-occurrence method, cite the knowledge graph and network construction workflow.
- If you use the website architecture, cite the static digital humanities platform design and presentation method.
